# Editable PDF iPhone App

This workspace contains the first implementation scaffold for a SwiftUI-based iPhone app that converts images into editable PDFs.

## Included
- SwiftUI app shell
- Liquid-glass style landing page
- Image import and camera capture flows
- Editable document model
- Background whitening preprocessing pipeline
- PDF export service
- Backend analysis contract models
- Branding notes

## Next step
Create an Xcode iOS app target and add these sources, or wire them into an existing app project.
